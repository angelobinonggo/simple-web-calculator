# Premium Minimalist Web Calculator

A responsive, gorgeous, and minimalist calculator application with a clean modern aesthetic, featuring rounded buttons, soft pastel colors, and Spotify-green accents. Built using vanilla HTML5, CSS3, and JavaScript.

![Calculator UI Showcase](https://raw.githubusercontent.com/angelobinonggo/simple-web-calculator/main/screenshot.png) *(Note: Add your actual screenshot here!)*

## ✨ Features
- **Basic Arithmetic Operations**: Addition (`+`), subtraction (`-`), multiplication (`×`), and division (`÷`).
- **Interactive UI**: Elegant glassmorphism/flat UI with micro-hover animations and elastic scale transitions on button click.
- **Percent Operator**: Dynamic percentage calculation support (`%`).
- **Dynamic Deletion**: Backspace support to delete mistyped characters easily (`DEL`).
- **Precision Floating Point Calculations**: Automatic rounding to avoid standard computer/JS precision errors (e.g. `0.1 + 0.2 = 0.3`).
- **Keyboard Shortcuts**: Fully functional keyboard support for desktop users.
- **Responsive Web Design**: Perfectly scaled for mobile, tablet, and desktop screens.

## ⌨️ Keyboard Shortcuts
Maximize your productivity by using standard keyboard mapping:
| Action | Keyboard Key |
|---|---|
| Numbers | `0` - `9` |
| Decimal Point | `.` |
| Addition | `+` |
| Subtraction | `-` |
| Multiplication | `*` |
| Division | `/` |
| Percentage | `%` |
| Calculate (Equals) | `Enter` or `=` |
| Delete (Backspace) | `Backspace` |
| Clear All (AC) | `Escape` |

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/angelobinonggo/simple-web-calculator.git
   ```
2. Open `index.html` directly in any web browser, or serve it using a local server:
   ```bash
   npx http-server -p 8000
   ```
3. Navigate to `http://localhost:8000` in your browser.
